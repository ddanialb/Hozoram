# Modabber Homework Bot

ربات استخراج تکالیف از سایت مدبر آنلاین

## نصب

```bash
npm install
```

## اجرا

```bash
npm start
```

## توضیحات

این ربات به صورت خودکار:
1. به صفحه جستجوی مدرسه می‌رود
2. مدرسه "هنرستان پسرانه هفتم تیر" را جستجو می‌کند
3. سال تحصیلی 1404-1405 را انتخاب می‌کند
4. با یوزرنیم و پسورد وارد می‌شود
5. تکالیف را از صفحه homework استخراج می‌کند

## خروجی

تکالیف در فایل `homeworks.json` ذخیره می‌شوند و شامل:
- اسم درس
- موعد تحویل
- خواسته (توضیحات)
- تاریخ

## نکته

این ربات از Puppeteer استفاده می‌کند که یک مرورگر سبک Chromium را دانلود می‌کند (فقط در اولین اجرا).
مرورگر به صورت visible اجرا می‌شود تا بتوانید فرآیند را ببینید.
